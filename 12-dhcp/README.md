# DHCP (Dynamic Host Configuration Protocol)

> **Status:** ✅ Completed

---

## 📖 Overview

Dynamic Host Configuration Protocol (DHCP) is a network protocol that automatically assigns IP addresses and other network settings to devices connected to a network.

DHCP eliminates the need to manually configure IP addresses on every device.

---

## 🎯 Why is DHCP Needed?

DHCP helps to:

- Automatically assign IP addresses
- Reduce manual configuration
- Prevent IP address conflicts
- Simplify network management
- Save time in large networks

---

## 🌐 DHCP Process (DORA)

1. **Discover** – Client searches for a DHCP server.
2. **Offer** – DHCP server offers an available IP address.
3. **Request** – Client requests the offered IP address.
4. **Acknowledge** – DHCP server confirms the IP assignment.

---

## 📚 DHCP Components

- DHCP Client
- DHCP Server
- DHCP Pool
- IP Address Lease

---

## 🌍 Real-World Example

When you connect your laptop or mobile phone to a Wi-Fi network, the router automatically assigns an IP address using DHCP.

---

## 📦 Advantages

- Automatic IP assignment
- Easy to manage
- Prevents duplicate IP addresses
- Saves administrator time

---

## ⚠️ Disadvantages

- DHCP server failure affects new devices.
- Less control over automatic IP assignments.
- Requires proper DHCP server configuration.
