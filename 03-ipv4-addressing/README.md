# IPv4 Addressing

> Status: ✅ Completed

---

## 📖 Overview

IPv4 (Internet Protocol Version 4) is the most widely used addressing system in computer networks. Every device connected to a network requires a unique IPv4 address to communicate with other devices.

An IPv4 address consists of 32 bits and is represented using four decimal numbers separated by dots.

Example:

192.168.1.10

---

## 🎯 Why is IPv4 Important?

IPv4 provides a unique identity for every device connected to a network. Without IP addresses, communication between computers, routers, servers, and mobile devices would not be possible.

---

## 🌐 IPv4 Address Format

An IPv4 address consists of four octets.

Example:

192.168.1.10

Each octet ranges from 0 to 255.

---

## 🏠 Public IP Address

A Public IP Address is assigned by an Internet Service Provider (ISP).

It is used to communicate over the Internet.

Example:

8.8.8.8

---

## 🏢 Private IP Address

Private IP Addresses are used inside local networks.

Examples:

192.168.x.x

172.16.x.x

10.x.x.x

---

## 🔄 Loopback Address

The loopback address is:

127.0.0.1

It is used to test the local computer's network stack.

---

## ⚠ APIPA Address

If a computer cannot obtain an IP address from a DHCP server, Windows automatically assigns an APIPA address.

Range:

169.254.0.0 – 169.254.255.255

---

## 🚪 Default Gateway

A Default Gateway allows devices to communicate with other networks.

Without a gateway, communication is limited to the local network.

---

## 🌍 Real-World Example

When I connect my laptop to a Wi-Fi network, it receives an IPv4 address. This address allows my laptop to communicate with other devices and access the Internet through the router.
