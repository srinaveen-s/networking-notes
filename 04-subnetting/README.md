# Subnetting

> Status: ✅ Completed

---

## 📖 Overview

Subnetting is the process of dividing one large network into multiple smaller networks called subnets. It helps improve network performance, security, and efficient use of IP addresses.

---

## 🎯 Why is Subnetting Important?

Subnetting reduces network congestion, improves security, and allows better management of devices within a network.

---

## 🌐 Network Address

The Network Address identifies the entire network.

Example:

192.168.1.0

---

## 📢 Broadcast Address

The Broadcast Address is used to send data to every device in the network.

Example:

192.168.1.255

---

## 👥 Host Address

Host addresses are assigned to devices inside the network.

Example:

192.168.1.10

192.168.1.20

---

## 📐 CIDR Notation

CIDR represents the subnet mask using a slash notation.

Example:

192.168.1.0/24

---

## 🌍 Real-World Example

A company has different departments such as HR, Finance, and IT.

Instead of using one large network, subnetting allows each department to have its own smaller network, making communication more organized and secure.
